# 🎯 TRADING DASHBOARD - QUICK REFERENCE CARD

## LOGIN (First Time Only)

```
1. Go to: https://yourusername.github.io/trading-dashboard
2. Click "Request OTP"
3. Enter your email
4. Check Gmail for code
5. Paste 6-digit code
6. Click "Verify OTP"
✅ You're in!
```

---

## ADD A TRADE (30 seconds)

```
1. Click "+ Add Trade"
2. Fill:
   • Symbol: NIFTY / TCS / etc
   • Buy or Sell (toggle)
   • Entry Date (today)
   • Entry Price: 24000
   • Quantity: 1
   • Exit Price: 24100 (auto-calculates P&L)
3. Click "Save Trade"
✅ Done! Trade appears instantly
```

## P&L FORMULA

```
P&L = (Exit - Entry) × Quantity × Leverage

Example:
Entry: ₹100
Exit: ₹105
Qty: 10
Lev: 2

P&L = (105 - 100) × 10 × 2 = ₹100 ✅ WIN
```

## FILTERS (Find Specific Trades)

```
Date:      From: ___   To: ___
Side:      All / Buy / Sell
Result:    All / Win / Loss / Breakeven / Open
```

**Examples:**
- Show June wins: Set dates + "Win"
- Show all Buy trades: Select "Buy"
- Show open trades: Select "Open"

## QUICK STATS FORMULA

### Win Rate
```
Wins / Total Trades × 100 = %
Example: 8 wins / 10 trades = 80% ✅
```

### Average Win
```
Sum of all wins / Number of wins
Example: ₹4000 / 8 wins = ₹500 avg ✅
```

### Profit Factor
```
Total Profit / Total Loss
Example: ₹4000 / ₹1000 = 4.0 ratio ✅
(Higher is better)
```
